# CSS

- 텍스트 관련   
  + color: ~ - 색깔 지정   
  + font-size: ~ - 크기 지정   
  + text-align: ~ - 텍스트 정렬   
  
- 여백 생성   
  + margin: ~   
  + padding: ~   
     
- 그리드
  + grid-template-columns 속성을 이용하여 그리드 설정   
    
- 박스
  + display:inline-block 으로 설정하여 박스 설정   
    
- 반응형 디자인   
  + @media(condition) {script} 조건에 따라 구문 실행   
   
- css파일 분리 및 불러오기   
  + &lt;link rel="stylesheet" href="style.css">를 (주로 head태그 내에)배치하여 css파일을 불러옴
